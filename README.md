# Journey by a Ship

This is a C++ computer graphics project that consists of three distinct sections or pages. These three pages include:

1. **Dockyard:** This is where the ship begins its journey.

2. **Urban City:** A depiction of an urban environment.

3. **Panama Canal:** A representation of a section of the Panama Canal.
